# Isaac-like Web Game

这是一个类以撒（Top-down Shooter）网页游戏，具有Roguelike元素、道具组合、Boss战等特性。

## 操作指南 (Controls)

### 桌面端 (Desktop)
- **移动**: `WASD`
- **攻击**: `鼠标左键` (射向鼠标位置) 或 `方向键` (固定八向射击)
- **冲刺**: `SPACE` (空格键)
- **主动道具**: `E` (需先拾取主动道具)
- **背包**: `B` (查看已获得道具)
- **图鉴**: `G` (查看怪物/道具信息)
- **帮助**: `H`
- **暂停**: `P` 或 `ESC`

### 移动端 (Mobile)
- **移动**: 左侧虚拟摇杆
- **攻击**: 右侧拖拽瞄准（自动射击）
- **操作按钮**: 界面上的功能按钮

## 游戏特性 (Features)
- **随机地牢**: 每次游戏都会生成不同的地图布局。
- **道具系统**: 
  - **被动道具**: 提升属性或改变攻击方式（如子弹追踪、分裂、激光）。
  - **主动道具**: 充能型技能（如无敌、全屏炸弹），按 `E` 使用，清理房间进行充能。
- **Boss战**: 每个关卡都有独特的Boss挑战。
- **难度系统**: 动态难度，随着层数增加敌人会变得更强。
- **小地图**: 实时雷达显示已探索区域，支持显示隐藏要素（需道具）。
- **图鉴系统**: 记录你遇到的所有道具和怪物的详细数据。

## 如何运行
1. 直接在浏览器中打开 `index.html`。
2. 或者使用 VS Code 的 "Live Server" 插件运行。

## 开发
- 修改 `js/game.js` 中的参数可调整游戏难度。
- 游戏素材为程序化生成，无需下载图片。
